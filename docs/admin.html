Refresh odds from The Odds API · 由 The Odds API 拉最新賠率

Free tier: 500 requests/month. One refresh = 1 request. Pulls all available WC2026 matches, takes the median odds across UK + EU bookmakers, matches them to your fixtures by team name + kickoff time, and shows a preview. You confirm before write.

Need a key? Sign up at the-odds-api.com (no credit card), then paste it into docs/js/firebase-config.js's ODDS_API_KEY.
